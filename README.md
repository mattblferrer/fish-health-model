# Fish Stress Model

## Description
This repository uses an ESP32-based recording setup to record fish-generated audio in (default) 10-second intervals, saved to an SD card through a microSD card adapter module. The audio is sent to a Pytorch Torchaudio trained model, trained on the dataset in the "data" folder, along with recordings generated by the recording setup. Some sample recordings are saved in the "record_set" folder.

## Specifications
The components used for the recording setup were: 
- 6x piezoelectric discs as hydrophones
- 2x WEMOS D1 R32 (ESP-32 based WiFi and Bluetooth microcontroller)
- 2x Arduino MicroSD card adapter module
